PHP haqida ma'lumot

PHP (/pi:.eɪtʃ.pi:/; inglizcha: PHP: Hypertext Preprocessor — „hipermatnli protsessor“) — skript tili boʻlib, web serverda oʻzgaruvchan HTML fayllarini yaratishda qoʻllaniladi.[1] 1994-yilda kanadalik dasturchi Rasmus Lerdorf tomonidan yaratilgan.[2] Hozirgi kunda deyarli barcha hosting-provayderlar mazkur tilni qoʻllab-quvvatlaydi. Odatda, PHP maʼlumotlar bazasi bilan ishlatiladi.

Assalomu alaykum hozirgi kunda eng mashxur dasturlash tillaridan biri bu PHP hisoblanadi chunki PHP juda ko'p frameworklar ishlab chiqarilgan va hozirgi kunda ko'p mashxur tizimlar PHP da ko'tarilgan. PHP dasturlash tiliga o'tishingizdan oldin siz HTML, CSS va JavaScript haqida tushunchaga ega bo'lishingiz zarur bo'ladi.PHP "PHP: Hypertext Preprocessor" ni qisqartmasi hisoblanadi.PHP yuqori dasturlash tillari sirasiga kiradigan, ochiq kodli skrip tilidir. PHP skriptlari server tamonidan bajariladi. PHP yuklab olish va undan foydalanish bepul.
